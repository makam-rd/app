# SI DUKA - Portal Pemakaman Umum "Raden Djaelani"

SI DUKA adalah sistem informasi digital untuk pengelolaan data dan administrasi
Pemakaman Umum "Raden Djaelani".

Aplikasi ini dirancang untuk membantu pengelola dalam mengatur data pemakaman
secara terstruktur, efisien, dan aman.

## Fitur Utama

- Pencarian data makam
- Manajemen data jenazah dan ahli waris
- Pengelolaan blok dan petak makam
- Manajemen pengguna dan hak akses
- Dashboard dan statistik
- Cetak laporan administrasi
- Sistem autentikasi dan keamanan data

## Versi

Siap digunakan untuk kebutuhan operasional.
